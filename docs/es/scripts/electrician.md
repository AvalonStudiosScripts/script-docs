# avalon_electrician

Script para el **trabajo de electricista** en FiveM. Los jugadores reparan cajas eléctricas mediante minijuegos con tres niveles de dificultad.

**Versión:** 1.0 | **Autor:** Mino

---

## Niveles de Dificultad

| Nivel | Nombre | Spots | Pago | Penalización |
|-------|--------|:-----:|------|:------------:|
| `1` | Simple | 33 | 100–150 tickets | 10 tickets |
| `2` | Media | 25 | 200–300 tickets | 25 tickets |
| `3` | Difícil | 15 | 400–600 tickets | 50 tickets |

## Herramientas

```lua
CONFIG.toolItem    = 'toolbox'
CONFIG.toolItemPro = { name = 'toolbox_pro', multiply = 1.3 }
```
