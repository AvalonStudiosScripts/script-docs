# avalon_electrician

Script para o **trabalho de eletricista** no FiveM. Os jogadores reparam caixas elétricas com minijogos em três níveis de dificuldade.

**Versão:** 1.0 | **Autor:** Mino

---

## Níveis de Dificuldade

| Nível | Nome | Spots | Pagamento | Penalização |
|-------|------|:-----:|-----------|:-----------:|
| `1` | Simples | 33 | 100–150 tickets | 10 tickets |
| `2` | Médio | 25 | 200–300 tickets | 25 tickets |
| `3` | Difícil | 15 | 400–600 tickets | 50 tickets |

## Ferramentas

```lua
CONFIG.toolItem    = 'toolbox'
CONFIG.toolItemPro = { name = 'toolbox_pro', multiply = 1.3 }
```
