# avalon_mining

Script para o **trabalho de mineiro** no FiveM. Mesma estrutura que `avalon_lumberjack` mas adaptada para minas.

**Versão:** 1.0 | **Autor:** Avalon

---

## Configuração

```lua
Lang = "Português"

Config.Loot = {
    { name = "rock",   label = "Rocha",  chance = 20, scoreGame = {min=1, max=5} },
    { name = "iron",   label = "Ferro",  chance = 5,  scoreGame = {min=1, max=5} },
    { name = "copper", label = "Cobre",  chance = 8,  scoreGame = {min=1, max=5} },
}
```
