# avalon_electrician

สคริปต์**งานช่างไฟฟ้า**ใน FiveM ผู้เล่นซ่อมกล่องไฟฟ้าด้วยมินิเกมที่มี 3 ระดับความยาก

**เวอร์ชัน:** 1.0 | **ผู้พัฒนา:** Mino

---

## ระดับความยาก

| ระดับ | ชื่อ | จุดงาน | รายได้ | ค่าปรับ |
|-------|------|:------:|--------|:-------:|
| `1` | ง่าย | 33 | 100–150 ตั๋ว | 10 ตั๋ว |
| `2` | ปานกลาง | 25 | 200–300 ตั๋ว | 25 ตั๋ว |
| `3` | ยาก | 15 | 400–600 ตั๋ว | 50 ตั๋ว |

## เครื่องมือ

```lua
CONFIG.toolItem    = 'toolbox'
CONFIG.toolItemPro = { name = 'toolbox_pro', multiply = 1.3 }  -- ลดเวลา 30%
```
