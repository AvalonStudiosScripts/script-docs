# avalon_mining

สคริปต์**งานขุดแร่**ใน FiveM มีโครงสร้างเดียวกับ `avalon_lumberjack` แต่ปรับสำหรับเหมือง

**เวอร์ชัน:** 1.0 | **ผู้พัฒนา:** Avalon

---

## การตั้งค่า

```lua
Lang = "ไทย"

Config.Loot = {
    { name = "rock",   label = "หิน",    chance = 20, scoreGame = {min=1, max=5} },
    { name = "iron",   label = "เหล็ก",  chance = 5,  scoreGame = {min=1, max=5} },
    { name = "copper", label = "ทองแดง", chance = 8,  scoreGame = {min=1, max=5} },
}
```
